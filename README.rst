Dependencies
------------

* Python >= 3.4

* pyyaml
* paramiko

------------
Installation
------------

* ``pip`` command may need ``sudo``

+-------------+---------------------------------------+
| Operation   | Command                               |
+=============+=======================================+
| Install     |``pip install .``                      |
+-------------+---------------------------------------+
| Upgrade     |``pip install --upgrade .``            |
+-------------+---------------------------------------+
| Uninstall   |``pip uninstall easy-ftp`              |
+-------------+---------------------------------------+

* Check the installed version: ``easy-ftp --version``
